# 编程作业3：复杂三维模型显示

## 作业要求：
- 支持以下三维模型文件格式中至少1种：OBJ，3DS，STL；
- 支持多个光源的光照效果；
- 支持多种视点浏览方式；

### 作业成果：
#### 功能:
- 以经典的泰迪熊模型为展示（OBJ格式）；
- 鼠标左键可拖动模型；
- 鼠标滚轮键可拖动缩放；
- 鼠标右键可平移模型；
- 作业最终的效果演示见Demo
